# Ejercico: Formulario de Registro para un taller de Programación

## Problema a Resolver

Una academia de tecnología necesita una página web donde los estudiantes puedan registrarse para participar en un taller de programación.

Actualmente, los registros se hacen por mensajes informales y esto genera:
- Información incompleta
- Datos desorganizados
- Dificultad para contactar a los principiantes

Se requiere una página Web HTML que contenga un formulario que permita recopilar estructuradamente la información de los estudiantes.
